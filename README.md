# embedded-touch-piano
대학생 임베디드시스템 설계 시간 자유 과제로 만든 Linux 기반 임베디드 Touch 피아노입니다.

<br/>

# 사용 기술
- C, C++
- Linux, QT

<br/>

# 기능
- 임베디드 보드 LCD 화면에 1옥타브(도~시)의 건반이 표시되고 터치 시 음을 출력합니다.
- 7개의 푸쉬버튼을 통해 7가지 음색을 선택할 수 있습니다.

<br/>
